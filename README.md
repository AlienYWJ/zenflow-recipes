# Zenflow Recipes 🌊

> The Open-Source Cognitive Infrastructure.

Welcome to the global action library for **Zenflow**. This is where we collect, share, and evolve automation recipes that compress cognitive load into a single click.

## What is a Recipe?
A Recipe is a declarative JSON file that tells Zenflow how to handle your thoughts. It transforms dirty text into clean Markdown, extracts YouTube transcripts, or syncs your ideas directly to your local Obsidian vault—all driven by your own API keys.

## How to Contribute
You don't need to be a hardcore developer. If you know how to write a good AI Prompt, you can create a Recipe.

1. **Fork** this repository.
2. Duplicate the `template.json` and rename it (e.g., `youtube-distiller.json`).
3. Fill in your JSON logic and system prompts.
4. Submit a **Pull Request**.

Our automation engine will review your submission. Once merged, it becomes instantly installable globally.

## Philosophy
- **No bloat:** Recipes should do one thing perfectly.
- **Local-first:** Data belongs to the user, not the cloud.
- **Calm tech:** Automation should feel like magic, not programming.
